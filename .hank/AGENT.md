## Build & Run

Install: `npm install`
Run: `./hank_loop.sh` (or `hank` if globally installed)

## Validation

Run these after implementing to get immediate feedback:

- All tests: `npm test`
- Unit tests only: `npm run test:unit`
- Single test file: `bats tests/unit/<file>.bats`
