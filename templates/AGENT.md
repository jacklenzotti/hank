# Agent Build Instructions

## Build & Run

Install: `npm install`
Build: `npm run build`
Run: `npm start`

## Validation

Run these after implementing to get immediate feedback:

- Tests: `npm test`
- Typecheck: `npx tsc --noEmit`
- Lint: `npx eslint src/`
